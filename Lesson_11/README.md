\# Lesson 11: MTS Tests with Allure Reports



\## Project Description

Automated tests for MTS website with Allure reporting framework. Tests verify online payment functionality.



\## Features

\- Allure annotations for detailed test reporting

\- Screenshot attachments on test steps

\- Step-by-step test execution tracking

\- Severity levels and test categorization



\## Test Scenarios

1\. Main page load verification

2\. Payment form filling and submission

3\. Payment information validation

4\. Different phone number formats testing

5\. Different payment amounts testing



\## Technologies Used

\- Java 11

\- Selenium WebDriver 4.15

\- JUnit 5

\- Allure 2.24

\- WebDriverManager

\- PageObject Pattern

\- Maven
