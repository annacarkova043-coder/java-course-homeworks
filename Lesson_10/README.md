\# Lesson 10: MTS Payment Page Tests with PageObject Pattern



\## Project Description

Automated tests for MTS website payment block using PageObject pattern. Tests verify placeholders for different service tabs and payment form validation.



\## Test Scenarios

1\. \*\*Service Tabs Placeholder Tests\*\* - Verify placeholders for:

&nbsp;  - Connection services (Услуги связи)

&nbsp;  - Home internet (Домашний интернет)

&nbsp;  - Installment (Рассрочка)

&nbsp;  - Debt (Задолженность)



2\. \*\*Payment Process Tests\*\* - Verify:

&nbsp;  - Navigation to payment form

&nbsp;  - Payment amount and phone number display

&nbsp;  - Payment form placeholders

&nbsp;  - Payment system icons presence

&nbsp;  - Complete payment flow validation
