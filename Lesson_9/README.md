\# Lesson 9: MTS Website Automated Tests



\## Project Description

Automated tests for MTS website (mts.by) focusing on the "Online payment without commission" block.



\## Test Scenarios

1\. \*\*Verify block title\*\* - Check that the block has correct title

2\. \*\*Verify payment system logos\*\* - Check presence of Visa, MasterCard, Belkart, Maestro logos

3\. \*\*Verify "More about service" link\*\* - Check link functionality

4\. \*\*Fill form and test Continue button\*\* - Test with phone number 297777777



\## Technologies Used

\- Java 11

\- Selenium WebDriver 4.15

\- JUnit 5

\- WebDriverManager

\- AssertJ

\- Maven
