# Java Course Homeworks

This repository contains homework assignments for the Java programming course.

## Structure

### Lesson 3: Park with Attractions
**Location:** `Lesson_3/AttractionPark/`

A Java program demonstrating:
- Class composition with inner classes
- Object-oriented programming principles
- Time management using `LocalTime`
- Collection management with `ArrayList`

**Files:**
- `park/Park.java` - Main Park class with inner Attraction class
- `Main.java` - Demonstration program

**To compile and run:**
```bash
cd Lesson_3/AttractionPark
javac park/Park.java Main.java
