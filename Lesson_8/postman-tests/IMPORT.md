# How to Import and Run Tests 
 
## Импорт коллекции в Postman 
1. Открыть Postman 
2. Нажать кнопку **Import** в левом верхнем углу 
3. Выбрать вкладку **File** 
4. Нажать **Upload Files** и выбрать файл `Postman_Echo_Tests.json` 
5. Нажать **Import** 
 
## Запуск тестов 
1. В коллекции **Postman Echo Tests** открыть папку **Request Methods** 
2. Запустить каждый запрос по отдельности, нажимая **Send** 
3. Для запуска всех тестов сразу: 
   - Нажать правой кнопкой на папку **Request Methods** 
   - Выбрать **Run folder** 
   - Нажать **Run Request Methods** 
 
## Просмотр результатов 
- После каждого запроса перейти на вкладку **Test Results** внизу 
- Все тесты должны быть зелеными (?)
