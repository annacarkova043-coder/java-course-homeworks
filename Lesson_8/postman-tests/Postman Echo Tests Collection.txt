Postman Echo Tests Collection
=============================

Автор: [Царькова]
Дата: [19.02.2026]

Описание:
Коллекция автотестов для API Postman Echo, проверяющая все методы из папки Request Methods.

Содержание:
1. GET Request - проверка GET метода с query параметрами
2. POST Request - проверка POST метода с отправкой JSON
3. PUT Request - проверка PUT метода
4. PATCH Request - проверка PATCH метода
5. DELETE Request - проверка DELETE метода

Каждый запрос содержит тесты для проверки:
✓ Код ответа (200 OK)
✓ Время ответа (< 500-600ms)
✓ Структуру JSON ответа
✓ Соответствие отправленных данных
✓ Заголовки ответа
✓ URL

Все тесты успешно проходят, подтверждая корректную работу Postman Echo API.

Инструкция по импорту:
1. Открыть Postman
2. Нажать "Import" в левом верхнем углу
3. Выбрать файл Postman_Echo_Tests_Царькова.json
4. Нажать "Import"
5. Открыть коллекцию и запустить тесты
