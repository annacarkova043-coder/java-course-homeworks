# Postman Echo API Tests 
 
## Описание задания 
В рамках Lesson 8 были написаны автотесты для коллекции Postman Echo. 
 
## Структура 
- `Postman_Echo_Tests.json` - экспортированная коллекция с автотестами 
- `README.md` - описание работы 
 
## Тестируемые методы 
1. GET Request - проверка GET метода с query параметрами 
2. POST Request - проверка POST метода с отправкой JSON 
3. PUT Request - проверка PUT метода 
4. PATCH Request - проверка PATCH метода 
5. DELETE Request - проверка DELETE метода 
 
## Запуск тестов 
1. Импортировать коллекцию в Postman 
2. Открыть папку Request Methods 
3. Запустить каждый запрос по отдельности или через Collection Runner
